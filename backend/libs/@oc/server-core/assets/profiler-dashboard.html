Profiler Metrics Explained

  • Min/Max Latency: Fastest and slowest response times observed.
  • Average Latency: Mean response time across all calls.
  • P95 Latency: “95% of our users get this speed or better.” It tells you the experience for almost everyone. It filters out the worst 5% of slow requests.
  • Cache Hit Rate: Percentage of requests served from cache vs database. Higher is better for performance.
  • Error Count: Number of failed requests. High error rates indicate reliability issues.
  • AsyncLocalStorage: Node.js feature used to track metrics per request without blocking the response thread.
